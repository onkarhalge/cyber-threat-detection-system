# Real-Time Cyber Threat Detection, Classification & Reporting System

## Setting Up the ML Models

This repository does not include the trained model files (they're too large for GitHub). Before running the project, download them separately:

1. Open the models folder on Google Drive:
   👉 https://drive.google.com/drive/folders/1uSsK6T3EPZbZ1z7Oq2dBLn9Wtvdda_cL?usp=sharing

2. Download all files and folders from the link above.

3. In the project root, create a folder named `models` (if it doesn't already exist).



## Steps to Run the Code

**Step 1 — Open PowerShell in the project folder

**Step 2 — Activate the virtual environment**
venv\Scripts\activate

You should see `(venv)` appear in your prompt.

**Step 3 — Run the app**
python run.py


**Step 4 — Wait for it to load**
You should see:

Loading classification models...
All models loaded successfully
Running on ...


**Step 5 — Open in browser**
